# 📄 Informe de Memoria de Título

Este repositorio contiene el **informe de memoria de título** desarrollado en formato **LaTeX**.  
El documento se encuentra organizado en múltiples secciones y capítulos, con su respectiva bibliografía y anexos.

---

## 📂 Estructura del Proyecto

```text
├── main.tex               # Archivo principal de compilación del informe
├── preambulo/             # Paquetes, estilos y comandos personalizados
├── capitulos/             # Secciones y capítulos del informe
│   ├── 01_Introduccion.tex
│   ├── 02_Marco_Teorico.tex
│   ├── 03_Metodologia.tex
│   ├── 04_Resultados.tex
│   ├── 05_Discusion.tex
│   └── 06_Conclusiones.tex
├── anexos/                # Material complementario y anexos
├── figuras/               # Imágenes y diagramas
├── tablas/                # Tablas en formato .tex o .csv
└── referencias.bib        # Archivo BibTeX con referencias
