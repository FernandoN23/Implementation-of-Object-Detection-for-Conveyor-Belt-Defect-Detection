# Implementación de Algoritmos de Reconocimiento de Objetos para la Detección de Fallas en Correas Transportadoras
Proyecto centrado en la creación y recopilación de un conjunto de datos sobre fallas comunes en correas transportadoras mediante imágenes, utilizadas para entrenamiento, validación y pruebas de modelos de reconocimiento de objetos basados en aprendizaje profundo (Deep Learning). Incluye la programación y la implementación de cuatro algoritmos de reconocimiento de objetos (YOLOv11, SSD300, DETR, DINO) y su evaluación mediante métricas empleadas en visión computacional.
